El componente **Menu** `<ui-menu>` es un contenedor estructural avanzado diseñado para agrupar y organizar elementos de lista, creando menús de navegación, listas de opciones o paneles de selección.

A diferencia de otros componentes, `<ui-menu>` actúa como un **wrapper** ***(envoltorio)*** que estiliza y agrupa a otros componentes **(`<ui-list>`, `<ui-menu-title>`, `<ui-menu-subtitle>`)**, centralizando el layout en un solo lugar.

## ¿Cuándo debería usarse?
Usa este componente cuando quieras:

- Menús Desplegables (Dropdowns): Listas de opciones que aparecen al hacer clic en un botón.
- Navegación en Paneles: Menús de configuración o laterales.
- Listas de Comandos: Agrupación de acciones relacionadas (ej. Opciones de perfil: "Editar", "Cerrar Sesión").

## Composición y Estructura
El componente `<ui-menu>` funciona mediante la inyección directa de los siguientes componentes estructurales:

| Componente        | Función                               | Atributo Clave                                           |
|-------------------|----------------------------------------|-----------------------------------------------------------|
| <span style={{whiteSpace: "nowrap"}}>`<ui-list>` </span>    | El elemento principal de acción. Emite el evento de clic. | `text`, `description`, `start-element`, `end-element`     |
| <span style={{whiteSpace: "nowrap"}}>`<ui-menu-title>` </span>  | Encabezado principal.                  | N/A (usa slot o atributo `text` interno)                 |
| <span style={{whiteSpace: "nowrap"}}>`<ui-menu-subtitle>` </span> | Subtítulo para agrupar secciones.   | `divisor` (opcionalmente `true` para una línea de separación) |

<Aside type="tip">Visita la documentación del componente **List** `<ui-list>` para conocer más sobre el componente implementado.</Aside>

## Variante de Diseño `type`
El único atributo que controla el `<ui-menu>` es `type`, que ajusta la densidad del padding interno para diferentes contextos de uso.

| Atributo type | Contexto de Uso                  | Características                                                                 |
|---------------|----------------------------------|----------------------------------------------------------------------------------|
| <span style={{whiteSpace: "nowrap"}}>`list` </span>         | Listas de Datos/Paneles          | Padding Compacto. Diseñado para listas densas o comandos donde el espacio vertical es importante. |
| <span style={{whiteSpace: "nowrap"}}>`navegation` </span>    | Menús de Navegación/Lateral      | Padding Generoso. Aplica un padding ligeramente mayor para mejorar el espacio táctil y la jerarquía visual, común en menús de aplicación. |

> **El ancho predeterminado del componente `<ui-menu>` es del 100%.**
Por diseño, el menú ocupará todo el espacio horizontal de su contenedor padre ***(width: 100%)***. Esto ofrece la máxima flexibilidad, ya que el menú se adaptará perfectamente a barras laterales o dropdowns.

<Aside type="tip">Para evitar que el menú se extienda por toda la página, es altamente recomendable implementarlo siempre dentro de un contenedor padre que defina el ancho deseado.</Aside>

## Disparador de eventos
El componente `<ui-menu>` **NO dispara eventos propios**. 

La lógica de acción es manejada íntegramente por los elementos hijos. El desarrollador solo necesita escuchar los eventos de los componentes `<ui-list>` que se encuentran dentro del menú.
