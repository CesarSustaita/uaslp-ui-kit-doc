La tipografía es un punto principal para asegurar la legibilidad, la jerarquía visual y la identidad institucional en nuestras interfaces.

### Open Sans como fuente principal

La selección de **Open Sans** como la fuente principal. Es una tipografía sans-serif reconocida por su excelente claridad y neutralidad.

- **Accesibilidad**: Su diseño amplio y abierto es ideal para la lectura en pantalla, asegurando una buena experiencia incluso en tamaños pequeños y para usuarios con diferentes capacidades visuales.
- **Disponibilidad**: Es una fuente fácil de obtener, lo que garantiza que la implementación en cualquier entorno sea sencilla.


### Implementación automática

El uso de la tipografía es **automático e inmediato en tu proyecto**, gracias a la arquitectura del UI Kit.

Al ejecutar `import 'uaslp-ui-kit';`, la fuente **Open Sans**, junto con todas sus características, se **inyecta automáticamente** en el head del documento. No es necesario agregar enlaces externos ni reglas `@font-face` en tu proyecto. La tipografía está **lista para usarse globalmente.**  

Si no utilizas un token específico, el texto de cuerpo (body) en tu proyecto adoptará automáticamente el **estilo base** ***(Regular, 16px)***.


### Escala Tipográfica 

La escala tipográfica sigue una escala modular para establecer una jerarquía visual clara. Existen tres grandes intenciones de uso para cada tamaño ideal para la contrucción de tu proyecto:

- **Encabezados (Headings)**: Para títulos y subtítulos funcionales para páginas principales o páginas de texto general para mantener una jerarquía visual de fácil lectura. 
- **Texto de Flujo (txt)**: Para párrafos y contenido de lectura que necesitan un espaciado relajado (150%).
- **Texto de Componente (body)**: Para elementos de interfaz (botones, labels) que necesitan un diseño compacto (espaciado de 100%).

## Uso y aplicación de tokens
La tipografía de tu proyecto está completamente unificada gracias a la arquitectura del UI Kit. Al importar la librería, todos los **estilos** y **valores** de la escala **tipográfica** se inyectan automáticamente como **variables CSS globales**.

Esto te da dos formas potentes de construir tu interfaz:

- **Tokens Primitivos:** Acceso al valor exacto ***(ej. solo el tamaño: `font size: var(--font-size-h1)`).***
- **Tokens Semánticos ***(Shorthand)***:** Estilos completos que combinan familia, tamaño, peso y altura de línea en una sola variable ***(ej. `font: var(--text-h1)`).***

A continuación, se muestra el listado completo de ambos tipos de tokens para su referencia y uso.

## Tokens Primitivos
Estos tokens definen los valores atómicos de la tipografía y son la base de los tokens semánticos.

<Aside type="tip" title="Modo de aplicación">
Puedes usar los tokens primitivos de la siguiente manera, ajustándolo a las necesidades para la creación de tu proyecto. 

export const css = `
    .mi-css-personalizado-para-texto {
    /*Uso de tokens primitivos de fuente*/
      font-family: var(--font-open-sans);
      font-size: var(--font-size-body-md);
      font-weight: var(--font-weight-bold);
      line-height: var(--line-height-text);

    /*Mediente el uso de tokens primitivos puedes ajustar 
    y crear el formato de tipografía como lo necesites*/

    /*Uso de tokens primitivos de color*/
      color: var(--text-default-default);
    }`;
export const cssName = 'ejemplo.css';

<Code code={css} lang="css" title={cssName} />

</Aside>

## Tokens Semánticos (Uso de Shorthand)
Estos tokens combinan todos los valores anteriores en una sola ***variable `font: var(...);`***, facilitando la aplicación del estilo completo con fidelidad de Figma.

<Aside type="tip" title="Modo de aplicación">
Puedes usar los tokens semánticos únicamente con una **línea corta** ***(Shorthand)*** donde contiene toda la **especificación** necesaria para el **estilo predeterminado** realizado en Figma. 

<Tabs>
export const css2 = `
    .mi-css-personalizado-para-texto {
    /*Uso de tokens semantico de fuente*/
      font: var(--txt-base);
    /* Esta única variable contiene: 
    Tamaño de fuente, Line Height y Tipografía*/

    /* Puedes ajustar el Line Height usando los valores primitivos*/
      line-height: var(--line-height-text);

    /*Uso de tokens primitivos de color*/
      color: var(--text-default-default);
    }`;
export const cssName2 = 'ejemplo.css';


export const css3 = `
    .mi-css-personalizado-para-texto {
    /*Uso de tokens semantico de fuente*/
      font: var(--txt-h4);

    Tamaño de fuente, Line Height, Tipografía, Peso de fuente y Tipografía (Open Sans predeterminada)*/

    /*Uso de tokens primitivos de color*/
      color: var(--text-default-default);
    }`;
