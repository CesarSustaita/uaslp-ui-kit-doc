### ¿Qué permite esta licencia?

La licencia MIT te otorga la libertad de:

- Usar el software para cualquier propósito _(personal, educativo o comercial)_.
- Distribuir copias del software _(original o modificado)_. 
- Fusionar este código con otros proyectos. 
- Vender copias del software _(siempre que se incluya el aviso de copyright)_.

La única obligación que impone esta _licencia_ es que debes incluir el aviso de derechos de autor _original_ y el _texto de la licencia_ en cualquier copia sustancial del software.


### Guía de Uso
Para asegurar la estabilidad y coherencia del _UASLP UI Kit_, se han establecido los siguientes lineamientos profesionales de uso:

1. **Integridad Visual y Estandarización**

El valor principal de este sistema es la consistencia. Al implementar estos componentes en aplicaciones universitarias o derivadas:

- Adherencia a Tokens: Utiliza siempre las variables de diseño _(colores, espaciados, tipografía)_ proporcionadas.
- Evita _"Hard-coding"_: Abstente de sobrescribir estilos con CSS forzado _(!important)_ a menos que sea una excepción crítica. Mantener la integridad visual fortalece la identidad institucional.

2. **Flujo de Trabajo Oficial**

Para garantizar que diseño y desarrollo esten en sintonía:

- Diseño: Utiliza exclusivamente la librería oficial de Figma más reciente para prototipar.
- Desarrollo: Consume el paquete oficial desde NPM. Evita copiar y pegar archivos sueltos manualmente, ya que perderás las nuevas funcionalidades.

3. **Reconocimiento de Autoría**

El desarrollo de software abierto se basa en el crédito. No elimines los avisos de copyright del código fuente. Si creas un sistema derivado, es una buena práctica profesional mencionar: _"Basado en la arquitectura del UASLP UI Kit"_

4. **Contribución**

Se fomenta la colaboración sobre la fragmentación. Si encuentras un error o desarrollas una mejora:

- Reporta y Contribuye: Ponte en contacto y envía tu corrección en lugar de mantener una versión privada corregida.
- Beneficio Común: Al contribuir garantizas que todos los futuros proyectos de la universidad se beneficien de tu hallazgo.
